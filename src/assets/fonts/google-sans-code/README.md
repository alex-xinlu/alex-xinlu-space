These font files are vendored for Astro's local font provider so builds do not
depend on `fonts.google.com` or `fonts.gstatic.com`.

Source family: Google Sans Code
Downloaded from: `https://fonts.gstatic.com/`
Configured in: `astro.config.ts`

Included variants:
- 300 normal / italic
- 400 normal / italic
- 500 normal / italic
- 600 normal / italic
- 700 normal / italic
