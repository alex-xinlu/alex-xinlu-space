These font files are vendored for Astro's local font provider so builds do not
depend on `fonts.google.com` or `fonts.gstatic.com`.

Source family: Noto Sans SC
Downloaded from: `@fontsource/noto-sans-sc`
Configured in: `astro.config.ts`

Included variants:

- 300 normal
- 400 normal
- 500 normal
- 600 normal
- 700 normal

Each variant includes `woff2` for browsers and `woff` for dynamic OG image
generation.
